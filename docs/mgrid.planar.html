planar¶

A class for planar graph corresponding to a multilayer network.

class mgrid.planar.PlanarGraph(dg=None)[source]¶

Bases: networkx.classes.digraph.DiGraph

Model multilayer graph in plane by contracting inter-edges.

All the edges left are intra-edges, so they must be associated with some layer.

There are two kinds of nodes, inter-nodes and planar nodes.

Note

If an inter-node is isolated in some layer, it cannot be recognised directly.

inter_nodes¶

all the inter-nodes, with two columns, “upper” and “lower”.
